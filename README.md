📊 Employee Salary & Experience Analysis Dashboard
This project showcases an interactive **Power BI dashboard** designed to analyze employee data, focusing on key metrics such as **salary, experience, age, and gender distribution**.
The dashboard provides meaningful insights through a combination of visualizations, helping users understand workforce patterns and trends effectively.

🔍 Key Features
* **Gender Distribution Analysis** – Visual representation of employee count by gender
* **Experience Insights** – Breakdown of total experience years across employees
* **Salary Analysis** – Displays total salary distribution and trends by age
* **Interactive Filters** – Dynamic filtering by gender and salary range
* **KPI Metrics** – Highlights total salary (35M) and employee count

📈 Visualizations Used
* Bar Chart for gender count
* Pie Chart for experience distribution
* Line Chart for salary trends by age
* Gauge Chart for total employee count
* Slicers for interactive filtering

💡 Purpose
The goal of this project is to demonstrate how **data visualization tools like Power BI** can transform raw data into actionable insights, supporting better decision-making in HR and business analytics.

🛠️ Tools & Technologies
* Power BI
* Data Modeling
* Data Visualization
